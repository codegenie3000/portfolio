Projects

Classroom Genie

I co-founded Classroom Genie which helps K-12 teachers crowdfund to get Gift Cards from Amazon.com and Walmart so they can get more supplies for their classroom.

Classroom Genie Web App

The web application was built in MeteorJS and utilizes MongoDB. For its front end, it uses Bootstrap and Spacebars (a derivative of Handlebars). Functionality includes:

  • Automated creation of a user's crowdfunding campaign
  • Payment processing
  • Promo code management
  • Email notifications and marketing using Foundation Emails
  • Cron jobs

It also includes various NPM API integrations including Stripe and Mailgun.
